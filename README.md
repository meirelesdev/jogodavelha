# Jogodavelha

> Jogo da velha (Tic-Tac-Toe) classico desenvolvido com HTML, CSS e JavaScript puro, com verificacao de vencedor e reinicio de partida.

## Stack

- HTML5
- CSS3
- JavaScript

## Prerequisitos

- Nenhum prerequisito especifico

## Como executar

```bash
# Clone o repositorio
git clone https://github.com/meirelesdev/jogodavelha.git
cd jogodavelha
```
